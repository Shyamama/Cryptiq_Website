# My App

A React + Vite application.

## Prerequisites

- Node.js and npm installed

## Setup

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Open the local URL printed by Vite (usually `http://localhost:5173`).

## Build

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
